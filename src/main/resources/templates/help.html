Методы API сервера - Мониторинг сети

GET-запросы

GET /

Мониторинг пинга серверов. Возвращает страницу с информацией о статусе пинга подключенных ПК.

GET /api/ping

Просмотр ответов в JSON формате. Возвращает данные о пингах в структурированном JSON-объекте для программной обработки.

POST-запросы

POST /api/setPC?computerName='NameComputer'

Добавить ПК для мониторинга. Передайте имя компьютера в параметре computerName для добавления его в список мониторинга.

POST /api/removePC?computerName='NameComputer'

Удалить ПК из мониторинга. Укажите имя компьютера в параметре computerName для исключения его из списка.

POST /api/clearAllPingData

Очистить таблицу мониторинга. Удаляет все накопленные данные о пингах, сбрасывая историю мониторинга.

Методы API сервера - БД MSSQL

GET-запросы

GET /api/sizeTable/MSSQL

Просмотр ответов в формате JSON данных. Возвращает данные о размерности БД в JSON-объекте для программной обработки.

GET /api/sizeTable

Просмотр результата роста БД MSSQL в разрезе таблиц. Возвращает страницу с информацией о размерности БД в графическом виде.

Примечание

Все POST-запросы требуют корректных параметров. Убедитесь, что имя компьютера в кавычках передано правильно. Для безопасности рекомендуется использовать HTTPS и аутентификацию при работе с API.
